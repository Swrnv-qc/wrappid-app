## Getting Started

1. Install all the npm module(s)

```
npm i
```

2. Setup wrappid web package

```
npm run wrappid:web:setup
```

3. Setup wrappid mobile package

```
npm run wrappid:mobile:setup
```

4. Change to Application Build Directory

- Web

    ```
    cd .\temp\web
    ```

- Mobile

    ```
    cd .\temp\web
    ```

5. Run Application

```
npm start
```

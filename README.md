## Getting Started

1. Install all the npm module(s)
```
npm i
```
2. Create wrappid web package 
```
npm run web:create
```
3. Create wrappid mobile package 
```
npm run mobile:create
```
4. Change to Application Build Directory
  - Web
    ```
    cd .\temp\web
    ```
  - Mobile
    ```
    cd .\temp\web
    ```
5. Run Application
```
npm start
```
